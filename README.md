# Khan Store - แอปมือถือ React Native

ยินดีต้อนรับสู่ **Khan Store** แอปอีคอมเมิร์ซที่สร้างด้วย **React Native** และ **Tailwind CSS** แอปนี้มีดีไซน์สะอาดและรองรับหน้าจอหลายขนาด ตั้งแต่มือถือจนถึงแท็บเล็ตและเดสก์ท็อป

## ฟีเจอร์
- **แสดงสินค้า**: จัดเรียงสินค้าในกริดแบบตอบสนอง (Responsive Grid)
- **หมวดหมู่สินค้า**: กรองสินค้าตามหมวดหมู่
- **แบนเนอร์โปรโมชั่น**: แสดงโปรโมชั่นปัจจุบันด้านบน
- **ไอคอนรถเข็น**: มีไอคอนรถเข็นที่แตะได้ (ยังเป็นตัวอย่าง)
- **ดีไซน์ตอบสนองทุกหน้าจอ**: ใช้ Tailwind CSS ทำให้หน้าจอสวยทั้งบนมือถือและแท็บเล็ต
- **แถบนำทางล่าง**: แถบนำทางสำหรับมือถือ มีไอคอน หน้าแรก, ค้นหา, รายการโปรด และโปรไฟล์

## ความต้องการเบื้องต้น

ก่อนรันแอปนี้ ต้องติดตั้งเครื่องมือต่อไปนี้:

- **Node.js**: เวอร์ชัน 14 ขึ้นไป
- **Expo CLI**: สำหรับพัฒนาและทดสอบแอป React Native
- **Yarn**: แนะนำสำหรับจัดการ dependencies

### ติดตั้ง Node.js
ดาวน์โหลดและติดตั้งได้จาก [Node.js](https://nodejs.org/)

### ติดตั้ง Expo CLI
